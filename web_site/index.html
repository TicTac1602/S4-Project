Pourquoi les QR Codes ?

Nous allons vous présenter un cours résumé à savoir pourquoi nous avons choisi un sujet concernant les QR Codes.

Un code QR, aussi appelé QR code (de l'anglais : quick response code, « code à réponse rapide ») est un type de code-barres en deux dimensions (ou code matriciel) constitué de modules noirs disposés dans un carré à fond blanc. L'agencement de ces points définit l'information que contient le code.

Les QR Codes sont très utilisés dans notre monde actuel. Il a été rapporté que 86,66% des utilisateurs de smartphones avaient scanné un code QR au moins une fois dans leur vie. Et 36,40% scannent au moins un QR Code par semaine. En plus de cela, 46,81% des répondants ont convenu que les codes QR facilitent la vie dans un monde sans contact. Mais ça ne s'arrête pas là. Un bon 38,99% des répondants souhaitent voir les codes QR utilisés plus largement à l'avenir.

Les QR Codes permettent par exemple de :

  • Naviguer vers un site internet, visiter un site web.
  • Regarder une vidéo en ligne ou un contenu multimédia.
  • Se connecter à une borne Wi-Fi.
  • Envoyer un courriel.

Les librairies utilisées

SDL 2.0

Simple DirectMedia Layer (SDL) est une bibliothèque logicielle libre. Son API est utilisée pour créer des applications multimédias en deux dimensions pouvant comprendre du son comme les jeux vidéo, les démos graphiques.

SDL_image

Cette librairie permet de charger des images, d'analyser un fichier pour connaître son format d'image.
